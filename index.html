Quién soy

Física nuclear,
IA y construcción

Soy estudiante de Ingeniería Física en la Universidad Nacional de Colombia, con enfoque en instrumentación nuclear, detección de radiación y electrónica aplicada. Combino el rigor de la física experimental con las herramientas más avanzadas de computación e inteligencia artificial.

Próximamente estaré en McMaster University como pasante de investigación Mitacs Globalink, trabajando en espectroscopía de aniquilación de positrones para la caracterización de películas delgadas a escala atómica.

Construyo startups y productos de software como solo founder, y divulgo ciencia, tecnología, IA y energía nuclear para hacer el conocimiento más accesible.

Contador Geiger de Bolsillo

Computación Cuántica 2025

Clasificador Cuántico Variacional para Identificación de Isótopos

Desarrollo de un Variational Quantum Classifier (VQC) para clasificar isótopos radiactivos a partir de datos de atenuación gamma. Fusión de computación cuántica con instrumentación nuclear experimental.

Computación Cuántica Machine Learning Python Qiskit

Ciencia de Materiales 2024

Análisis de Difracción de Rayos X e Identificación de Fases

Análisis estructural de materiales mediante datos XRD para identificar fases cristalinas. Procesamiento de difractogramas con QualX, indexación de picos y desarrollo de scripts Python para visualización y análisis comparativo.

XRD Python QualX Análisis de Datos

GamifyMyLife

Convierte tu vida en un RPG. Derrota tus tareas como si fueran bosses.
